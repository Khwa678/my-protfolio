# 🌐 Personal Portfolio Website

This is my personal portfolio website built to showcase my skills, projects, and experience in **Machine Learning**, **Web Development**, and **Frontend Engineering**.  
The website highlights my journey, certifications, and work while providing an easy way for recruiters and collaborators to connect with me.

## 🚀 Features

- Fully responsive and modern UI
- Clean and minimal design
- Sections for skills, projects, resume, and contact
- Mobile-friendly layout
- Smooth navigation
- Showcases ML & web dev projects


## 🛠️ Tech Stack

**Frontend:**  
- HTML5  
- CSS3  
- JavaScript  
- React (optional if used)

**Tools Used:**  
- VS Code  
- Git & GitHub  
- Canva/FlowCV (for resume)
